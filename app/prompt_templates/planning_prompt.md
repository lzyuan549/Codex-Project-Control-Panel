【后端实现强制规范】
1. MyBatis-Plus：
   - Service 接口必须继承 IService<Entity>。
   - Service 实现类必须继承 ServiceImpl<Mapper, Entity>。
   - Mapper 接口必须继承 BaseMapper<Entity>。
   - 严禁直接拼接 SQL；复杂查询优先使用 LambdaQueryWrapper，必要时使用 @Select。
2. 本地文件映射：
   - 必须通过 WebMvcConfigurer 配置 addResourceHandlers，使上传文件可通过 URL 访问。
   - 上传与回显能力要纳入接口设计；如涉及文件功能，必须明确 FileController 或同等职责模块。
3. 全局异常处理：
   - 必须使用 @RestControllerAdvice。
   - BusinessException 返回中文错误信息，状态码 400。
   - MethodArgumentNotValidException 返回 code/msg 中文提示。
   - 未知异常返回“系统繁忙，请稍后重试”，状态码 500。
   - 接口入参优先使用 @Valid + 校验注解；禁止在业务代码中大量手写 if 判空。
4. SQL 脚本：
   - db/ 下建表脚本使用 CREATE TABLE IF NOT EXISTS。
   - 表名、字段名统一使用下划线命名法。
   - 字段必须带中文 COMMENT。
   - schema.sql 或等效初始化脚本需放在 db/ 目录。
